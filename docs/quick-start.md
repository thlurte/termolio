# Quick Start

## Prerequisites
- Node.js (v20.19.0+ or v22.12.0+)
- npm or yarn

## Installation
1. Clone the repository:
```bash
git clone <repository-url>
cd termolio
```
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
4. Open your browser and navigate to `http://localhost:5173`
